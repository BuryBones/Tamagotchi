Инструкция по сборке:

1. Склонируйте ветку master. В директории окажутся: 
 - Папка src
 - pom.xml
2. Через командную строку, запустите в этой директории "mvn package"
3. Создастся папка "target". Перейдите в неё в командной строке.
4. Среди прочего, вы должны увидеть файл .jar
5. Выполните команду "java -jar <название jar архива>"
   Например "java -jar tamagotchi-1.0.jar".

Про игру:

	Питомцы:

Питомцы имеют две основные характеристики: здоровье и веселье.
Повышение уровня здоровья достигается кормлением. Повышение уровня веселья достигается игрой.
Если уровень здоровья питомца падает до 0, питомец погибает.
Некоторое время (5 минут) после гибели питомца появление нового - невозможно.

	Выбор питомцев:

В игре присутствуют три типа питомцев: Квадратный, Круглый и Треугольный.
Характеристики питомцев абсолютно одинаковые, различается только внешний вид и тип корма.
Если покормить питомца неподходящим кормом, то он получит гораздо меньше очков здоровья,
чем если бы был покормлен подходящим.
Со временем питомец вырастает. Питомцы имеют три стадии роста: ребёнок, подросток, взрослый.
Рост происходит при кормлении, если достигнут определённый уровень здоровья.
Уровень веселья не рост не влияет.

Каждая стадия роста занимает: 2 часа

	Сохранение:

Сохранение происходит при выходе из игры автоматически.
Загрузка сохранённой игры происходит автоматически.
При отстутствии сохранённой игры - создаётся новая.
При выключенной игре прогресс продолжается.
